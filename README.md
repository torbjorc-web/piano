## Features

- Play piano notes with mouse clicks.
- Play piano notes with computer keyboard keys.
- Visual feedback when keys are pressed.
- Happy Birthday song box with step controls.
- Audio playback using `.wav` files.

## How to Run Locally

1. Download or clone the repository.
2. Open `index.html` in your browser.
3. Make sure the `audio/` folder is in the same directory as the HTML file.

## GitHub Pages Setup

1. Push the project to a GitHub repository.
2. Go to **Settings**.
3. Open **Pages**.
4. Choose **Deploy from a branch**.
5. Select the `main` branch and `/ (root)` folder.
6. Save and wait for the site to publish.

## Notes

- Audio file names must match the names used in `script.js`.
- If sound does not work, check that the `audio/` folder is included in the repository.
- If keys look misaligned, verify that `style.css` is the active stylesheet.
